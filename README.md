# Anime girl to email

Using nodemailer to sent an anime girl picture to your email.

**Sneak peek of what you will receive**
![](/docs/images/sample.png)

## Stack

- Next.js
- Node.js
- TypeScript
- Nodemailer
- Jest

## Setup

- Install dependencies

```bash
pnpm install
```

- Start server

```bash
pnpm api
```

- Start web

```bash
pnpm web
```
